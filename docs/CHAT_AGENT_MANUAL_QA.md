## B. Agent Speech

| # | Шаг | Ожидание | ✓ |
|---|-----|----------|---|
| B1 | Режим **Agent Speech**, TTS включён в Settings | После ответа — озвучка, затем idle / снова микрофон | |
| B2 | Тот же чат, режим **Agent**, TTS включён | Ответ **без** озвучки | |
